== Description ==

Lauro Socializer plugin is a social share plugin  for social network ( Twitter, Facebook, Google+) and email subscription (newsletter) function. Enjoy IT!

A few notes about the sections above:

 
*   You can change the background to any color you want
*   Display Facebook Like thumbnail on your blog sidebar.
*   Automatically display the social share buttons.
*   Show your LinkedIn and Google+ profile. 
*   Google +1 share option for each post. Easily visitors share your blog post.


For more information visit [Lauro Socializer plugin page](http://all-inclusiv-web/lauro-socializer "Lauro Socializer plugin page")

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Download this lauro-socializer.zip file to your computer.
1. Go to Plugins and click Add New to upload and activate the plugin.
1. Then, go to Widgets section and drag-and-drop the Lauro Socializer to the sidebar.
1. Simply enter your Feedburner, Twitter, and Facebook ID in. Click Save. That`s all .
